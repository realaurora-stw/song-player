How It Works

1

Download Your MP3

Get the MP3 file of the song you want to create synchronized lyrics for.

You can use any MP3 file. For best results, try to find a clean version with minimal background noise. The clearer the vocals, the better the transcription will be.
2

Generate AI Transcript

Upload your MP3 to ElevenLabs for AI transcription with word-level timestamps.

1. Go to https://elevenlabs.io/app/speech-to-text
2. Upload your MP3 file
3. Wait for transcription to complete
4. Click "Export" and select JSON format
5. Make sure "Include speakers" is NOT checked
6. Download the JSON file
3

Mix MP3 + Transcript

Use our Mixer tool to combine your MP3 and the JSON transcript into a .songlyrics file.

1. Open our Mixer tool
2. Upload both your MP3 and JSON files
3. The tool will automatically combine them
4. Download the resulting .songlyrics file
4

Play With Synchronized Lyrics

Open the .songlyrics file in our Player to enjoy word-by-word synchronized lyrics.

1. Open our Player tool
2. Upload your .songlyrics file
3. Enjoy your music with synchronized lyrics!
4. Try different audio effects like nightcore or reverb
